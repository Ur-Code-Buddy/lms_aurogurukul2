> ️Please give us a star ⭐️ on GitHub, so that it reaches more people.

## Why self host?

You should self host Aurogurukul, if you:

- want complete control of your data
- want to host it behind a firewall for internal use

## How do I self-host?

You have two options for self-hosting the Aurogurukul stack. MediaLit is our sister service that supports file uploads.

- **Option A**: Self-host both Aurogurukul and MediaLit.
- **Option B**: Self-host Aurogurukul and use hosted MediaLit.

![Self hosting options](/assets/schools/self-host.svg)

### Self host Aurogurukul

See [the self hosting guide](/self-hosting/self-host).

## Hosted version

If you don't want to self-host, Aurogurukul's hosted version is available at [Aurogurukul.app](https://aurogurukul.app). It is managed by the team behind Aurogurukul. We will take care of everything, just come with your team and content.
